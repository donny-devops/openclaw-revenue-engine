# GitHub Actions Debug Sprint

## What I Need
- Repository URL
- Workflow name or failing run URL
- Error log excerpt
- Expected outcome

## Deliverables
- Root cause summary
- Corrected YAML or patch plan
- Validation commands/checklist
- PR-ready summary

## Operator Checklist
- Preserve existing security permissions
- Avoid broad `contents: write` unless justified
- Prefer least-privilege tokens and deterministic checks
